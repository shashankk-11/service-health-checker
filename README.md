# Service Health Checker

A lightweight API health monitoring platform that checks website and API availability, records uptime history, and demonstrates DevOps practices such as CI/CD, containerization, and observability.

## Architecture

Two-tier system:

Client → Monitoring API → SQL Database

## Features

- Register services to monitor
- Perform health checks
- Record uptime history
- CI/CD pipeline
- Docker container support
